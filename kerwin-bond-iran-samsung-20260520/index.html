过去五个交易日，美国国债市场经历了自2025年4月以来最剧烈的抛售。10年期收益率从4.1%飙升至4.67%，30年期突破5.18%。PPI同比6.0%、CPI同比3.8%双双超预期，叠加霍尔木兹海峡封锁推动WTI站稳$100以上，市场从"年内降息"叙事急速切换至"加息概率浮现"。我观察的宏观对冲AI精选组合64只标的中，53只在5D窗口内录得下跌——但这是一次"贴现率冲击"，估值压缩而非盈利恶化。AI算力基础设施直接受益股（NBIS、ARM、MRVL）逆势上涨，证明市场在区分"谁能将AI需求转化为真实现金流"与"谁仅靠AI叙事支撑估值"。YTD维度下，组合中仍有50只标的录得正收益——绝大多数持仓的基本面没有受损。

我认为，这次是短暂的逆风。利率飙升的根源是地缘政治驱动的能源供给冲击，而地缘政治问题本质上是可逆的。2025年5月，10Y同样冲至4.6%，"滞胀论"四起，但9月之后Fed三连降，科技股暴涨。那些在恐慌中保持定力、甚至逆向操作的人，赚到了最大的一段收益。

这一轮利率飙升的传导链非常清晰：霍尔木兹海峡被封锁 → 全球原油航运受阻 → WTI突破$100 → 美国汽油破$4.50/加仑 → CPI 3.8% YoY、PPI 6.0% YoY双双超预期 → 降息预期归零 → 加息概率浮现。

2025年5月，几乎完全相同的故事发生过：关税冲击推高通胀预期，10Y冲至4.6%，市场认为降息遥遥无期。但关税是政治工具——可开可关。当政策逆转，10Y从4.6%回落至4.1%，Fed在9月后启动了三连降。那些在5月恐慌中坚持不卖的人，在Q4赚到了全年最大的涨幅。

2026年的"开关"不是关税，是海峡复航。两者形式不同，但本质一致：都是政治决策，不是经济结构变迁。如果2025年5月保持定力是对的，那么2026年5月拥有相同的逻辑基础。

五 · 伊朗博弈困境

封锁海峡正在成为战略负债

2026年5月14日，美国总统特朗普抵达北京，对中国进行为期三天的国事访问，与习近平主席会谈逾两小时，同游天坛。这是特朗普重返白宫后首次对华国事访问。紧随其后，俄罗斯总统普京于5月19日至20日接续访华。塞尔维亚总统亦即将到访。

京报网将这一安排描述为"美俄元首一周内前后脚访华，罕见外交安排"。中国人民大学重阳金融研究院的定调是：中国正在从国际秩序的"参与者"加速转向"塑造者"。

在这个格局下，伊朗的战略处境正在急剧恶化。

伊朗的博弈结构

策略 A：封锁霍尔木兹海峡
└─ → 全球 20% 原油航运受阻
└─ → WTI $103+，全球通胀
└─ → 危害全球普通民众既得利益 → 成为"逆天遭人恨"的对象

策略 B：威胁破坏海底电缆
└─ → 全球互联网面临风险
└─ → 进一步强化"全球公敌"标签

外部环境变化：
└─ 美、俄元首两周内先后访华 → 中国成为全球外交枢纽
└─ 作为"秩序塑造者"，中国不可能支持破坏全球公共品的行为体
└─ → 伊朗外交孤立加深 → 谈判压力增大 → 海峡复航概率上升

封锁海峡、威胁破坏海底电缆——这些行为危害的是全球普通人的既得利益：油价负担、航运成本、互联网连接。这不是在与某一个国家对抗，而是在与全世界的日常生活为敌。伊朗每多封锁一天，就多失去一份国际同情。

5月19日，CBS News报道：特朗普叫停了对伊朗的既定军事攻击，同时称"严肃谈判"正在进行。这不是软弱，而是对博弈均衡的理性判断：让伊朗自己消耗自己的国际信誉，比军事打击更有效、成本更低。

3月23日的历史先例支持这一判断：仅Trump的谈判言论就让布伦特原油从$114单日暴跌至$102。一旦海峡复航预期形成，油价可能在1-2周内回落至$80-90区间。届时，支撑当前利率恐慌的"能源通胀"基础将从源头瓦解。

六 · 三星工会博弈

牌已出尽，靴子落地

2026年5月20日，三星电子管理层与工会的最后谈判破裂。工会威胁展开18天罢工——据估算可能造成每天约$7亿美元的损失。消息传出后，三星股价盘中急跌，但随即反弹。

以博弈论的框架来看，这是一场"最后通牒博弈"，而工会已经打完了所有牌。

三星工会博弈 · 三轮演进

第一轮（5月12日）：工会提出永久性薪资结构改革要求
└─ 三星回应：仅提供一次性绩效金，拒绝任何永久性承诺
└─ 政府调解介入

第二轮（5月14日）：三星主动邀约重返谈判桌
└─ 姿态放低——会长罕见公开道歉（媒体描述为"千载难逢的一次"）
└─ 但底线寸步不让：薪酬制度灵活性不可让渡

第三轮（5月20日）：最后谈判破裂
└─ 工会：18天罢工威胁（$7亿/天）
└─ 三星：沉默克制，不接受结构性让步
└─ 盘中急跌 → 随即反弹 → 市场认可"靴子落地"

三星管理层的策略是典型的"承诺策略"：如果这一次接受永久性薪资改革，等于把定价权交给了工会。棘轮效应下，下一次索求只会更大。短期罢工损失可以计算，但永久性制度让步的代价无法估量。

而工会的致命弱点在于非对称成本。罢工工人损失工资，同时被非罢工同事眼红——他们损害了大多数人的利益。在全球投资者和产业链视角中，这部分人正在被贴上"贪得无厌"的标签。

更重要的是：罢工对三星的存储业务可能构成"反直觉利好"。当前全球存储供需极度不对称：AI拉动HBM需求爆发，SK海力士、三星、美光全部满产。存储行业的问题是，需求一来，供给立刻扩产，价格反被压制。罢工恰恰打破了"扩产→降价"的周期逻辑——它使供给持续处于收缩状态，强化市场对供给紧张的预期，倒逼价格和毛利率进一步提升。三星的存储业务占其利润大半，罢工造成的产能损失很可能被价格上涨带来的利润增量对冲甚至超过。

三星高层"不咸不淡地道了个歉"——管理层没有慌，很有定力。如果接受了工会的要求，反倒制造更大的不确定性：你不知道下一次更大的威胁是什么，又会什么时候爆发。而现在——谈判破裂、拒绝接受、靴子落地——牌已出尽。盘中急跌后反弹的走势，是市场在用真金白银确认这一判断。

NBIS（+10.4%）、ARM（+7.3%）、MRVL（+7.2%）在5D窗口中逆势上涨——当宏观恐慌来临时，资金不是逃离AI，而是从"AI叙事"向"AI现金牛"集中。持有真实AI现金流标的，是穿越利率风暴的最优策略。

04

黄金-4.63%、BTC-5.45%、ETH-7.53%——当所有对利率敏感的资产同步下跌，驱动因素是统一的（利率），而非各自基本面恶化。一旦海峡谈判出现正面进展、10Y回落至4.2%以下，这些资产将出现最猛烈的反弹。

05

海峡复航是最大的上行催化剂。日本6月若加息（77%概率）是短期扰动，但日本的加息恰恰是"经济复苏→正常化"的信号，中长期对日本股票是利好。当前日本股票的回调——6594.T、6324.T、6268.T——是买入机会。

八 · 结语

逆风中的定力

核心观点

当前美债收益率飙升的根源——霍尔木兹海峡封锁——是一个正在被大国协调所松动的变量。伊朗的外交孤立在加深，三星的工会博弈已走向终局。支撑利率恐慌的两根支柱都在松动。2025年5月的教训是：恐慌时离场的人错过了Q4的暴涨；恐慌时逆向买入的人赚到了全年最大的收益。这一次，我倾向于后者。

5月15日，我在组合交易日志中写下这样一段话——它恰好概括了这篇文章的全部逻辑：

"跌6.9%买入。市场情绪担忧通胀及加息可能，US10Y、US30Y大增。过去一两天这是宏观对普遍科技股的压制因素。不认为会长久。"

当利率冲击退潮——无论是通过海峡谈判取得进展、还是伊朗的外交孤立最终转化为实际行动、还是三星罢工靴子落地后存储价格开始反映供给收缩——被压缩的估值弹簧将释放。在逆风中保持定力，在恐慌中逆向操作，这就是我的选择。
