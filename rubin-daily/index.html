MLCC 上游材料扩展逻辑

Rubin / AI 服务器对高容、高可靠 MLCC 的增量需求，不只传导到 Murata、TDK、Samsung Electro-Mechanics 等成品厂， 也会继续向上游材料扩散。MLCC 结构中，钛酸钡粉主要用于介质层，镍粉/镍浆用于内部电极； 高容 MLCC 越薄、层数越多、可靠性要求越高，对粉体粒径、纯度、分散性和烧结一致性的要求越高。 因此本日报新增 MLCC Materials 观察组：钛酸钡重点看 4028.T、4092.T、4078.T、4100.T、300285.SZ； 镍粉/金属粉体重点看 5016.T、5713.T、5727.T。核心跟踪指标是高端 MLCC 交期、Murata / Taiyo Yuden / Samsung Electro-Mechanics 扩产、 钛酸钡粉体产能利用率、材料涨价函，以及 4028 / 4092 / 4078 电子材料分部利润率。

PCB 企业定位

本组重点不是简单比较 PCB 公司谁涨得多，而是区分产业链位置：板厂、载板/ABF、CCL 材料、A+H 高阶 PCB 平台。Rubin 的关键变化是高层数、高可靠、高速信号和机柜 Midplane 价值提升。

3037.TW 欣兴 / Unimicron

产业链位置：高阶 PCB + ABF 载板平台型龙头。独特性在于同时覆盖 Rubin 相关的 Midplane PCB 与 ABF 载板两条主线，是确定性最高的 PCB 映射之一。护城河来自高阶制程、客户认证、ABF 产能与良率经验；弱点是估值已开始反映确定性。

4958.TW 臻鼎-KY / ZDT

产业链位置：全球大型 PCB 平台厂，消费电子、服务器、汽车等多场景覆盖。独特性在于规模、客户覆盖和产能调度能力，若 AI server 高阶 PCB 订单放量，利润弹性可能高于更稳健的龙头。护城河是大客户认证、量产组织能力和多基地交付；风险在高阶 AI server PCB 的良率爬坡和实际份额。

002916.SZ 深南电路 / Shennan Circuits

产业链位置：国内 PCB + 封装基板 + 电子装联一体化平台。独特性不是直接替代欣兴或臻鼎，而是提供国产 AI 服务器链条中“PCB、载板、装联协同”的观察窗口。护城河来自通信与服务器客户基础、封装基板能力和系统级交付经验；关键看高端 AI 服务器订单占比。

2476.HK / 300476.SZ 胜宏科技 / Victory Giant

产业链位置：高阶 PCB 与 AI 服务器 PCB 高弹性标的，A+H 双上市口径。它在 PCB 领域的地位是中国高阶 PCB 扩产和 AI server PCB 订单弹性的代表，不是 ABF/Midplane 确定性最强的公司。独特性在于高多层板、HDI、高速板和海外客户放量带来的业绩弹性；护城河主要来自产能爬坡、客户认证和成本效率。需要特别跟踪产品结构：若 Rubin 增量更多集中在 Midplane / ABF，胜宏的绝对量可能增长，但相对份额和估值弹性要看其能否切入更高价值结构。

6213.TW 联茂 / ITEQ

产业链位置：CCL / 覆铜板材料层，不是 PCB 板厂。独特性在于它对应的是 Rubin PCB 从 M7 到 M8 等更高等级材料升级，属于“PCB 材料上游”而非板厂产能。护城河来自树脂配方、低损耗材料能力、客户认证和与高阶 PCB 厂的协同；关键看高速低损耗材料的产品 mix 和毛利率。

MLCC Materials 企业定位

钛酸钡粉 / 介质粉体

★ 4028.T Ishihara Sangyo：通过 Fuji Titanium 与 Murata 绑定，并参与 MF Material 扩产 MLCC 钛酸钡，是本轮材料上游映射最清晰的标的。

★ 4092.T Nippon Chemical Industrial：BESPA BT 直接用于 MLCC 介质层，高纯度、窄粒径分布和 Ba/Ti 摩尔比稳定是高端 MLCC 的关键指标。

★ 4078.T Sakai Chemical Industry：高纯介质粉体和水热合成路线，覆盖钛酸钡、钛酸锶、钡盐、二氧化钛，适合观察传统化工向电子陶瓷材料重估。

4100.T Toda Kogyo：湿法钛酸钡粉体供应商，适合作为 MLCC 材料链补充观察，但需要拆分业务占比。

300285.SZ 国瓷材料：国内钛酸钡、MLCC 配方粉和纳米粉体龙头之一，重点看国产替代与高端 AI 服务器客户穿透能力。

镍粉 / 镍电极粉

★ 5016.T JX Advanced Metals：先进金属材料平台，覆盖镍粉、半导体靶材、电磁屏蔽膜等电子材料；镍粉方向最值得优先跟踪。

5713.T Sumitomo Metal Mining：镍资源和金属材料体系完整，但公司体量大，MLCC 镍粉对利润弹性会被资源和冶炼业务稀释。

5727.T Toho Titanium：金属粉体和 MLCC 电极粉历史相关标的，需持续核查 JX Advanced Metals 整合及交易状态。
