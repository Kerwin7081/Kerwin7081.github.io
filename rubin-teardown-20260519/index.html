03 · Rubin架构

Rubin NVL72 是 rack-scale AI supercomputer

NVL72 核心配置包括 72 颗 Rubin GPU、36 颗 Vera CPU、20.7TB HBM4、54TB LPDDR5X、NVLink 6、ConnectX-9 与 BlueField-4。72 GPU 对 36 CPU 的配置说明，CPU 在 Agent 环境、工具调用、数据流动和 KV cache 管理中重新变成关键资源。

04 · CMX / STX / eSSD

约 100 万美元 3D NAND 存储的真正含义：NAND 开始进入推理热路径

CMX 是 NVIDIA 在 STX 架构下的 context memory storage platform，用于 long-context、multi-turn、agentic AI inference。它由 BlueField-4 storage processor 驱动，扩展 GPU memory，形成 shared pod-level context tier，并针对临时 KV cache 优化。

HBM 仍是 GPU 第一内存层，CMX/eSSD 承接 HBM 与 DRAM 放不下、传统存储又太慢的上下文外溢层。铠侠是明确参与者之一，但 Samsung、Micron、Solidigm、Phison 等也会在企业级 SSD、QLC eSSD、控制器、系统认证和 AI storage 方案里参与竞争。

图：NVIDIA CMX Context Memory Storage Platform。

结语：Rubin 是 AI 工厂的剖面图

Vera 交付 SpaceX 的推文，是一个很好的观察切口。它让市场重新看到，Agentic AI 需要更多 GPU，也需要更多 CPU、更大的 HBM、更快的上下文存储、更强的 DPU/NIC、更复杂的网络、更高密度的液冷和电力系统。
