2022 年的三重打击

2022 英伟达下跌不是单一原因，而是三重去化的叠加：去估值泡沫（美联储 3 月启动 40 年最猛加息，纳指全年 −33%）、去矿卡需求（BTC 从 $47K 跌至 $16K，以太坊 9 月完成 The Merge 转 PoS，GPU 挖矿需求断崖归零）、去渠道库存（2021 年挖矿+居家推高的销量透支需求，二手矿卡冲击游戏卡定价）。本质上是一次罕见的"需求侧+供给侧+分母侧"同时收紧。

从 GPT-1 到 GPT-5.5：OpenAI 十年路线图

2018-06 · GPT-1
1.17 亿参数，首个生成式预训练模型。以无监督预训练+有监督微调范式确立了 GPT 系列的技术路线。概念验证阶段。
2019-02 · GPT-2
15 亿参数，文本生成能力飞跃。OpenAI 以安全为由分阶段发布——这是"AI 太危险不能全放"叙事首次进入公共讨论。
2020-05 · GPT-3
1,750 亿参数，小样本学习突破。无需微调即能执行翻译、写作、代码生成等多任务。第一个让科技界开始严肃讨论 AGI 可能性的模型。
2022-11-30 · ChatGPT
基于 GPT-3.5 微调的对话式界面。5 天破百万用户，2 个月破亿（史上最快产品增速）。NVDA 当天从 $15.62 涨至 $16.91（+8.2%）。
2023-01 · 微软 $100 亿
微软宣布向 OpenAI 追加投资 $100 亿，标志着 hyperscaler AI 军备竞赛正式打响。
2023-03-14 · GPT-4
多模态（文本+图像输入），推理能力质变。律师考试前 10%，AP 全科高分。从 ChatGPT 到 GPT-4 仅 4 个月，从"对话玩具"到"专业工具"。
2024-05-13 · GPT-4o
原生多模态（音频+视觉+文本），毫秒级实时响应，免费开放。AI 交互进入语音/视觉原生时代。
2024-09 · o1 推理模型
首个思维链推理模型，数学/编程/科学突破。推理时代前奏——模型不再只是"生成"，而是"思考"。
2025-08 · GPT-5
统一系统，融合推理+对话+多模态。推理成本大幅下降。
2026-03 · GPT-5.4
前沿模型，覆盖推理、编程、Agent、工具使用。Agent 能力成为核心卖点。
2026-04 · GPT-5.5
最新旗舰：编码、研究、长程自主工作。首次明确瞄准"计算机使用"场景——AI 不再只是回答，而是做事。
关键节奏

GPT-1 到 GPT-3 用了 2 年完成基础研究。ChatGPT（2022.11）是真正的破圈时刻——首次把大模型以对话形式直接推向公众。GPT-4（2023.3）4 个月后就证明了这不是玩具。2025 年起模型进入爆发期，几乎每季度一个大版本，迭代速度从"年"压缩到"月"。2026 年的 GPT-5.5 明确以"计算机使用"为目标——AI 的竞争已从"能说什么"转向"能做什么"。

Q1 → Q2：觉醒时刻

2023 年 5 月 24 日，Q1 FY2024 财报数字本身并不好看——营收同比还 −13%，游戏仍在去库存。但 Jensen 在电话会上给了 Q2 指引 $11.0B——当时华尔街共识只有 $7.2B。这个 $3.8B 的指引差距，在半导体史上几乎没有先例。

驱动这个跳跃的不是当期业绩，而是 ChatGPT 上线后 5 个月内，所有 hyperscaler 同时意识到：他们必须在 GPU 集群上投入前所未有的资本开支，否则会在 AI 时代掉队。

8 月 Q2 财报 $13.51B vs 指引 $11.0B，再次大幅超标。Jensen 的判词成为行业共识："A new computing era has begun."

Q3 → Q4：引擎全速

Q3 营收 $18.12B（+206% YoY），单季利润超 FY2023 全年。Q4 收于 $22.1B（+265% YoY），数据中心已占营收 83%。毛利率从 Q1 的 64.6% 飙到 Q4 的 76.0%。从 $7.19B 到 $22.1B，四个季度营收翻 3 倍。

第五章 · 存储板块

Agent 推理时代的"新 GPU"：为什么存储比算力更重要

核心判断

训练时代的瓶颈是算力（GPU），推理时代的瓶颈是内存。每一个 Agent 行动——调用模型、检索上下文、执行工具——都受制于数据在 GPU 与 HBM 之间移动的速度。"Memory Wall"（内存墙）正在取代"Compute Wall"成为 AI 基础设施的第一约束。

物理现实：为什么推理是内存瓶颈

训练时 GPU 做大批量矩阵乘法，算力利用率可以推到 90%+。但 Agent 推理完全不同：每次行动需要加载模型权重（数十 GB）、用户上下文（数 MB 到 GB）、RAG 检索结果、工具输出——这些数据必须全部驻留在高带宽内存中。当 HBM 容量不够时，系统被迫在 GPU 之间搬运数据，昂贵的 GPU 算力在等待数据的时间里空转。

Forbes 2026 年 5 月的判断一针见血："AI 的瓶颈不是以 TB 为单位的存储容量——而是以 GB 为单位的 GPU 内存。"

HBM 的技术壁垒远高于普通 DRAM：需要 TSV（硅通孔）垂直堆叠 8-16 层 DRAM 芯片，与 GPU 通过 2.5D 封装（CoWoS）紧密耦合。这个制造复杂度意味着 HBM 的供给弹性极低——三星和 SK 海力士已警告 AI 驱动的存储短缺可能持续到 2027 年及以后，客户已提前数年预定产能。

类比差异

存储板块与 2022 年英伟达有一个关键区别：英伟达是"被低估的垄断者"（业绩还在跌），而存储巨头们当前业绩已创历史新高，市场已有一定定价。但叙事仍有巨大差距——市场仍然把存储当作"半导体周期股"，而没有把它当作"AI 推理基础设施的核心瓶颈"来定价。当 HBM 从 GPU 的"配件"升级为 Agent 推理的"第一约束"，存储的估值逻辑将从周期切换到成长。这个认知转换本身就是最大的 Alpha。

第六章 · 时代类比

Agent 推理时代：全面基础设施类比

重构后的排序

如果只能选一个方向作为"Agent 时代的英伟达"——存储 > 定制 ASIC > 光互连 > 边缘推理 > Agent 平台。推理的物理瓶颈首先是内存，然后才是互连带宽和算力架构。

01
💾 存储 / HBM：Agent 推理的第一物理瓶颈（类比信度 ★★★★★）

SK 海力士 (000660.KS) · 三星 (005930.KS) · 美光 (MU) · SanDisk (SNDK)
训练=算力瓶颈，推理=内存瓶颈。HBM 供给弹性极低（TSV 堆叠+CoWoS），产能已被锁定至 2027 年。海力士 Q1 利润 +405% YoY、美光 HBM 售罄数季、DRAM 价格持续暴涨——但市场仍以"周期股"视角定价，而非"AI 推理基础设施"。这是 2026 年最接近英伟达 2022 的结构性机会：高壁垒、低供给弹性、需求确定性极强、估值逻辑尚未切换。

AVGO
Broadcom：定制 ASIC 寡头（类比信度 ★★★★★）

Google TPU、Meta MTIA 靠 AVGO 设计。推理时代每个 hyperscaler 都需要优化过的推理芯片，定制 ASIC 切换成本极高——一旦设计定下来就是多年锁定。hyperscaler 自研 ASIC 的集体转向就是 AVGO 的"ChatGPT 时刻"。

COHR
Coherent：光互连物理瓶颈（类比信度 ★★★★）

推理时代数据移动量远超训练。800G→1.6T→3.2T 光模块升级路径。英伟达 Q1 13F 首次披露 780 万股 COHR 持仓——NVIDIA 自己也在下注光互连供应链。

LITE
Lumentum：光学产能锁定（类比信度 ★★★★）

EAML 产能锁定至 2027。Jefferies 目标价 $1,200。英伟达 $20 亿战略合作。若推理驱动光互连需求爆发，LITE 将享受产能溢价定价权。

ARM
ARM：边缘 Agent 生态基石（类比信度 ★★★★）

云端训练≈英伟达，边缘推理≈ARM。ARM v9 AI 扩展 + 终端推理需求 = 类比 2016 年 CUDA 生态早期。但"ChatGPT 时刻"尚未出现——需要全民级终端 Agent 产品引爆。

CRM
Salesforce：企业 Agent 货币化平台（类比信度 ★★★）

不是硬件"镐和铲"而是数据+工作流基础设施。Agentforce 拥有全球最大企业客户关系数据。但软件层护城河远不如物理层。

跨资产视角

Agent 时代有一个独特的变量：加密轨道。如果 Agent 之间需要自主交易、身份验证、价值结算——这恰好是 BTC/ETH/稳定币的用武之地。这不是"加密货币"叙事，而是"Agent 经济的基础支付层"。Agent 时代的基础设施可能是一组集合：存储（HBM）+ ASIC（AVGO）+ 光互连（COHR/LITE）+ 边缘推理（ARM）+ Agent 结算（加密轨道）。

结语

英伟达的故事告诉我们：最大的超额收益不在"当期业绩最好"的时候，而在"当期业绩还在跌，但叙事已经彻底改变"的时刻。

2022 年末的 NVDA：$14.60 的股价、−21% 的营收、矿难+加息双杀。但 Jensen 在下一季度的电话会上给出了 $11B 的指引——华尔街震惊，然后 4 个季度后营收 $22B。

Agent 推理时代正在寻找它的"2022 英伟达"。我们的结论是：如果只选一个方向——存储板块（HBM）。推理的物理瓶颈首先是内存带宽和容量，然后才是光互连和定制算力。海力士、美光、三星这三大 HBM 寡头处于结构性供给短缺中，但市场仍以"周期股"而非"AI 基础设施核心瓶颈"定价——这个认知差距就是 Alpha。

Agent 时代的基础设施不是一家公司，而是一组供应商的集合：存储（海力士/美光/三星）+ ASIC 设计（AVGO）+ 光互连（COHR/LITE）+ 边缘推理（ARM）+ Agent 结算层（加密轨道）。
