关键判断

NVIDIA 对模型公司的投资不是被动 LP 行为。每一笔投资的本质都是"资本前置 + 硬件后置"：先用投资锁定客户，再以算力合同确保收入回流。OpenAI 的 $1,000亿框架甚至被 NVIDIA 自己标注为"不保证完成"，但框架本身已是市场信令。

结语

NVIDIA 的 NVentures + 企业直投双轨体系，本质上是一个用资本锁定全栈 AI 生态的战略工具：

1. 基础模型层（7家）→ 投钱给客户 → 客户用钱买 GPU → $1,220 亿闭环
2. 算力基础设施层（5家）→ 投钱给 GPU 云 → 扩大算力供给 → 更多人用 CUDA
3. 应用层（8家）→ 投钱给应用 → 更多人消耗 GPU 推理 → 持久化芯片需求
4. 量子计算（2家）→ 投钱给后 GPU 技术 → 确保下一个范式仍有 NVIDIA 利益
5. 能源与物理（6家）→ 投钱给基础设施瓶颈 → 数据中心供电、光互连、机器人

最重要的判断：这不是传统意义上的 VC 投资组合优化。NVentures 的每一分钱都在为 NVIDIA 的硬件销售和平台锁定服务。理解这一点，就能理解为什么 NVIDIA 愿意在 OpenAI 身上签 $1,000亿框架——因为每一个投出去的美元，最终都以 GPU 收入的形式翻了数倍回来。

关注点：CoreWeave IPO 进程、Cursor 估值飙升背后 NVIDIA 的持股比例、Scintil Photonics + Ayar Labs 的光互连布局协同。
