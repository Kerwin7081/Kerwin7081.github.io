04 · HOW TO USE

你平时只需要做三件事

01

每周看一次动作

先看红、黄、绿状态和“当前动作”。没有催化剂或阈值变化时，不需要每天反复判断。

02

交易前看三条规则

确认是否达到加仓条件，是否触发减仓条件，是否出现证伪信号。三条规则比短期情绪更重要。

03

财报后让看板更新

财报公布后更新收入、毛利率、HBM、企业级SSD、库存和现金流，再调整状态与动作。

最简使用法：平时只看“持仓面板”和“催化剂”；准备交易时再打开“动作纪律”。
05 · DECISION BOUNDARY

这张看板能做什么，暂时不能做什么

所有阈值均标记为待Kerwin确认的监测草案，不是自动交易指令。公司基本面强化，也可能同时意味着证券层面的风险收益变差；网页将这两层判断分开呈现。
