在2026年5月13日至20日的五个交易日中，Kerwin 宏观对冲AI精选组合64只标的中，53只录得下跌，仅11只上涨。这不是单一公司或板块的问题——这是美国国债市场自2025年4月以来最剧烈的单周抛售引发的系统性贴现率重定价。

5月13日PPI同比6.0%（2022年以来最大涨幅）、5月15日CPI同比3.8%（2023年5月以来最高）——两次超预期通胀数据叠加霍尔木兹海峡封锁推动WTI原油站稳$100以上，彻底瓦解了市场对2026年任何降息的期待。CME FedWatch出现了结构性翻转：加息概率首次超过降息概率。Polymarket给出57%全年零降息定价，27%的加息概率虽已从周中36%回落，但远高于一个月前的个位数。

然而，组合中AI算力基础设施直接受益股（NBIS +10.4%、ARM +7.3%、MRVL +7.2%）逆势上涨，证明市场并未放弃AI产业趋势——它在区分"谁能将AI需求转化为真实现金流"与"谁仅靠AI叙事支撑估值"。

2025年复盘：政策冲击 → 政策逆转 → 利率回落

2025年初，通胀粘性出现，Fed推迟降息。4月初特朗普宣布"对等关税"，10Y一度暴跌至4.045%（市场解读为衰退恐惧 > 通胀风险）。一周后关税暂停，10Y反弹至4.5%。5月触及~4.6%高点——即当前被反复引用的"2025年5月以来最高"基准。

关键转折：关税是政策工具，可开可关。当关税威胁消退，通胀预期随之缓和，Fed得以在9月后启动连续3次降息（累计75bp），12月10日最后一次降息时将10Y压至4.15%。全年以10Y收于4.16%收官。

2026年：供给冲击 → 无简单出口 → 利率可能继续上行

这一次没有"关税暂停"这样的off-ramp。霍尔木兹海峡被封锁、WTI持续$100以上、美国汽油破$4.50/加仑——能源供给冲击驱动的结构性通胀无法通过货币政策直接解决。Natixis的John Briggs直言："解决战争，我们才能计算压力何时缓解。否则油价驱动的通胀粘性每天都在延长。"

2年期收益率突破Fed Funds上沿是最危险的信号——市场在说当前利率"不够紧"。如果6-7月CPI/PPI继续超预期，Fed加息概率将从27%继续攀升，届时10Y可能突破5%、30Y可能向5.5-6%迈进，引发更广泛的金融条件收紧。

七 · 五层框架总结

按 Kerwin 五层框架审视当前组合

第1层：全球流动性 / 贴现率（主导因子）

US10Y +57bp至4.67%，US30Y突破5.18%，日本10Y升至2.79%（29年高）。全球三大利率中心同步上行，无风险利率的全面抬升构成组合5D表现的首要解释变量。这是"分母驱动"的下跌——估值压缩而非盈利恶化。

第2层：制度 / 财政 / 政治方向

美伊僵局推高油价→第二波通胀→Fed加息路径重新定价。Fed领导层换血增加政策不确定性。$691B单周国债拍卖加剧供需失衡。日本BOJ鹰派转向（6月77%加息概率）引发全球利率联动。关税、战争、财政赤字三线交织。

第3层：技术 / 产业传导

AI产业趋势未变：NVIDIA与IREN签$34亿合同、NBIS手握Meta等大单持续增长、ARM受益端侧AI+CPU需求。但"蹭AI概念"的小市值标的正在被清洗。市场在区分"真实AI现金流"与"AI叙事估值"。

第4层：公司质量 / 估值

业绩miss的（LEU、6324.T）被无情惩罚；稀释融资的（IREN $3B CB）不被原谅；内部人减持的（POWL）放大抛压。市场已从"买一切AI"切换至"买能证明自己的AI"。

第5层：跨资产确认

黄金-4.63%、铜-4.83%、BTC-5.45%、ETH-7.53%——贵金属+工业金属+加密全线下跌。这不是单一资产错杀，是全球风险偏好系统性收缩。DXY +1.04%确认资金从风险资产向美元现金回流。

八 · 策略含义

面对"真滞胀"与"假通胀恐慌"的差异

核心结论

2025年是"假通胀恐慌"——政策冲击可逆，Fed有空间降息。
2026年很可能是"真滞胀序幕"——供给冲击难逆，Fed进退维谷。

当前环境对组合管理的关键启发：

1. 高YTD涨幅标的需警惕"回撤惯性"：SNDK（+483%）、7709.HK（+397%）、智谱AI（+654%）等已在过去5天领跌，说明极端涨幅后对利率逆转极度敏感。这不是减仓建议，但需要设定止损/止盈纪律。

2. AI算力基础设施是当前唯一"免疫区"：NBIS、ARM、MRVL的5D逆势上涨证明——当宏观恐慌来临时，资金不是逃离AI，而是从"AI叙事"向"AI现金牛"集中。这是去伪存真的健康过程。

3. 加密/贵金属/铜的同步下跌是预警信号：当传统避险资产（黄金）和数字黄金（BTC）同时下跌，说明流动性在系统性收缩。如果这一趋势延续，组合中的₿板块和🥇板块需要评估仓位耐受度。

4. 日本加息是全球利率上行的放大器：BOJ若在6月加息（77%概率），将推动JGB 10Y突破3%，日元套利交易进一步平仓，对全球风险资产施加额外压力。

5. 盯住霍尔木兹：这是目前所有宏观不确定性中最大的单一变量。如果海峡通航恢复，油价回落到$80-90区间，通胀预期将迅速降温，利率会跟随回落，当前最痛苦的持仓将迎来报复性反弹。
