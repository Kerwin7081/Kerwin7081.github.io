三层定价体系说明

为什么 GPU 租赁有三层价格？

Hyperscaler（超大规模云）—— AWS、Azure、GCP 的官方按需定价。代表「零售」算力价格天花板。GPU 实例通常捆绑 8 卡整机起租，附带宽、存储和 SLA。价格最高，但可用性最稳定。

NeoCloud（新型云厂商）—— CoreWeave、Nebius、Lambda Labs 等专业 GPU 云。代表「批发」算力市场。通常支持单 GPU 起租，无长期合约要求，价格约为 Hyperscaler 的 30-50%。是 AI 创业公司和中小团队的主力算力来源，也是算力价格竞争最激烈的层级。

Marketplace（现货市场）—— Vast.ai 等 P2P GPU 租赁平台。代表算力的「现货/边际定价」。个体数据中心和矿场挂牌闲置 GPU，按秒计费，无最低租赁时长。价格最低但可靠性参差——本系统对 H200 额外过滤 reliability < 0.90 的报价。

投资含义

三层价差（Hyperscaler vs NeoCloud vs Marketplace）不是套利机会，而是算力供应链价值分配的直接体现。价差扩大 = NeoCloud 价值捕获增强 → 利好 CRWV/NBIS；价差缩小 = 算力供给充裕或 Hyperscaler 降价抢量。三层之间的价差方向是判断算力供需松紧的前置指标。

产品说明

关于本监测系统

Kerwin GPU 算力租赁价格监测系统（Compute Monitor）是一款面向投资研究的算力定价追踪工具，由 Kerwin 团队设计、Enya.Clawd 自动化实现与运维。

核心能力

  • 四代 GPU 全覆盖：A100、H100、H200、B200，从上一代到最新 Blackwell 架构，完整覆盖算力代际迁移。
  • 三层定价体系：Hyperscaler（AWS/Azure/GCP 官方按需价）、NeoCloud（CoreWeave/Nebius/Lambda 等新型云厂商）、Marketplace（Vast.ai 等现货市场），揭示算力供应链的价值分配。
  • Provider 级穿透：追踪 10 家主要算力供应商的 GPU 型号定价，关联上市公司代码（CRWV/NBIS/DOCN/AMZN/MSFT/GOOGL），直接映射到公开市场标的。
  • 量化信号引擎：7 条实时信号覆盖 A100 反衰减、年度合约增长、Neocloud 价差扩大、B200 波动率、HBM 成本传导、Blackwell 延期、定制芯片渗透等关键主题。
  • MTD / 累计变动：对比月初及历史基准价格，提供月度变动（MTD）和累计变动百分比，辅助判断算力通胀/通缩趋势。

H200 特殊处理

H200 价格统计自动排除 reliability score < 0.90 的低质报价，防止 RunPod 等平台的异常低价扭曲中位数和价格区间。

CME 算力期货与监测指标意义

2026 年 5 月 12 日，CME Group 与 Silicon Data（全球交易公司 DRW 背书的 GPU 市场情报领导者）联合宣布将于 2026 年下半年推出全球首个标准化算力期货合约，待监管审查。该合约以 Silicon Data 每日 GPU 按需租赁基准价为基础，现金结算，覆盖 H100、H200 及后续代际。

本监测系统的指标意义：算力期货的推出标志着 GPU 计算从定制化协商投入正式过渡为标准化市场定价资产——相当于原油、电力、航运运费发展出期货市场的历史时刻。本系统追踪的三层 GPU 租赁价格，本质上是算力「现货」价格的实时快照。当 CME 期货上线后，现货-期货基差（spot-futures basis）将成为判断算力供需松紧的核心指标：期货升水（contango）预示供给宽松预期，现货升水（backwardation）则反映短期算力紧缺。
