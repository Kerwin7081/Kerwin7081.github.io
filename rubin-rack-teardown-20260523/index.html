三、PCB 细项拆解

PCB +233% 不是单一板卡涨价，而是 Compute board、Switch board、Midplane 和 ConnectX 相关板级价值同时抬升。其中 Midplane 是 Rubin / VR200 相对 GB300 的全新增量，从无到有，是理解 PCB 价值重估的关键。

PCB细项GB300VR200变化投资含义
Compute板22L HDI26L M8+115%层数与材料等级同步提升，考验高阶 HDI / M8 量产能力
Switch板24L32L+81%交换板复杂度提升，强化高阶多层板需求
Midplane无44L 新增+∞全新需求，从无到有；对具备高层数背板 / midplane 能力的供应商最关键
ConnectX×72$270/片+$19.4K网络互联相关板级价值扩张，验证 Networking 与 PCB 的交叉受益

个股映射：Unimicron / 欣兴是最确定的 Rubin PCB 受益者，原因是 Midplane 与 ABF 双线暴露；ZDT / 臻鼎-KY弹性更高，但执行风险也更高，关键在高阶 AI server PCB 订单兑现和良率爬坡；深南电路是国内少数同时覆盖 PCB、封装基板与电子装联的中游平台型标的，适合作为国产 AI 服务器 PCB / 载板 / 装联协同观察项；胜宏科技绝对出货量仍可能增长，但如果 VR200 增量更多集中在 Midplane / ABF 等特定结构，其相对份额存在被结构性稀释的风险，需要跟踪客户份额、产品结构和高层数板占比。

Power/Cooling 表中是机柜内 BOM 口径。机柜外还有更大的电力与热管理需求，包括 800V 电压架构、UPS、配电柜、母线槽、电源模块、CDU、冷板、快接头、热交换和整套液冷基础设施。冷板/液冷映射优先看 3653.TW 健策、3017.TW 奇鋐；2308.TW 台达电更适合作为电源 + 散热平台型标的；VRT 看数据中心电力基础设施，6805.TW 看水冷机构件与快接头。

六、公司简要定位

Memory · SNDK · SanDisk

NAND/存储周期高弹性标的。Rubin 内存成本抬升之外，AI 数据中心对高速存储和容量扩张也会形成外溢需求。

Memory · 285A.T · Kioxia / 铠侠

NAND 供应链核心标的。更偏存储侧映射，不是 HBM 纯标的，但适合观察 AI 数据中心存储资本开支扩散。

Memory · 000660.KS · SK hynix

HBM 与高端 DRAM 核心标的。Rubin 对内存成本的拉升让 HBM 供给、ASP 与良率成为最重要变量。

Memory · MU · Micron

美股内存代表。若 HBM 与高端 DRAM 进入供给约束周期，Micron 是美股中最直接的内存映射。

Memory · 005930.KS · Samsung Electronics

内存与先进封装生态参与者。观察点在 HBM 追赶、DRAM 价格周期和 AI 客户认证进度。

PCB · 3037.TW · Unimicron / 欣兴

产业链位置：高阶 PCB + ABF 载板平台型龙头。独特性是同时覆盖 Rubin 相关 Midplane PCB 与 ABF 载板两条主线，是确定性最高的 PCB 映射之一。护城河来自高阶制程、客户认证、ABF 产能与良率经验。

PCB · 4958.TW · ZDT / 臻鼎-KY

产业链位置：全球大型 PCB 平台厂，覆盖消费电子、服务器、汽车等多场景。独特性是规模、客户覆盖和产能调度能力；若 AI server 高阶 PCB 订单放量，利润弹性可能高。护城河是大客户认证、量产组织能力和多基地交付。

PCB · 002916.SZ · Shennan Circuits / 深南电路

产业链位置：国内 PCB + 封装基板 + 电子装联一体化平台。独特性不是替代欣兴或臻鼎，而是观察国产 AI 服务器链条中 PCB、载板、装联协同兑现。护城河来自通信与服务器客户基础、封装基板能力和系统级交付经验。

PCB · 2476.HK / 300476.SZ · Victory Giant / 胜宏科技

产业链位置：高阶 PCB 与 AI 服务器 PCB 高弹性标的，A+H 双上市口径。它在 PCB 领域的地位是中国高阶 PCB 扩产和 AI server PCB 订单弹性的代表，不是 ABF / Midplane 确定性最强的公司。护城河来自高多层板、HDI、高速板量产能力、客户认证、产能爬坡和成本效率。关键风险是 Rubin 增量若集中在 Midplane / ABF，胜宏绝对量会增长，但相对份额和估值弹性要看能否切入更高价值结构。

PCB材料 · 6213.TW · ITEQ / 联茂

产业链位置：CCL / 覆铜板材料层，不是 PCB 板厂。独特性是对应 Rubin PCB 从 M7 到 M8 等更高等级材料升级，属于 PCB 材料上游。护城河来自树脂配方、低损耗材料能力、客户认证和与高阶 PCB 厂协同。

MLCC · 009150.KS · Samsung Electro-Mechanics

韩国 MLCC 与电子组件标的。YTD 强势，反映市场已开始交易 MLCC 在 AI 机柜中的价值抬升。

MLCC · 6981.T · Murata

全球 MLCC 龙头。Rubin 供电密度提高后，高规格 MLCC 用量与单价是核心观察点。

MLCC · 6762.T · TDK

MLCC 与电子元件龙头，作为 Murata 之外的日股观察标的。适合验证被动元件景气是否扩散。

MLCC Materials · 4028.T · Ishihara Sangyo

最清晰的钛酸钡上游映射之一。其子公司 Fuji Titanium 与 Murata 绑定，并与 Murata / Ishihara 成立 MF Material 扩产 MLCC 钛酸钡；逻辑不是泛化工，而是高端 MLCC 材料供给。

MLCC Materials · 4092.T · Nippon Chemical Industrial

电子级钛酸钡材料供应商，BESPA BT 直接用于 MLCC 介质层。重点看高纯度、窄粒径分布、Ba/Ti 摩尔比稳定等指标是否被 AI 服务器高端 MLCC 放大。

MLCC Materials · 4078.T · Sakai Chemical Industry

高纯介质粉体与水热合成路线标的，覆盖钛酸钡、钛酸锶、钡盐、二氧化钛等电子材料。适合作为“传统化工股向电子陶瓷材料股重估”的观察对象。

MLCC Materials · 钛酸钡 · 4100.T · Toda Kogyo

湿法钛酸钡粉体供应商，适合作为 MLCC 介质粉体补充观察标的；关键要继续拆业务占比和客户结构。

MLCC Materials · 钛酸钡 · 300285.SZ · 国瓷材料

国内钛酸钡、MLCC 配方粉和纳米粉体龙头之一。重点看国产替代逻辑之外，是否能穿透到全球高端 AI 服务器 MLCC 客户链。

MLCC Materials · 镍粉 · 5016.T · JX Advanced Metals

★ 镍粉方向最核心跟踪企业。先进金属材料平台，覆盖镍粉、半导体靶材、电磁屏蔽膜等电子材料，不是纯 MLCC 镍粉股，但电子材料属性强。

MLCC Materials · 镍粉 · 5713.T · Sumitomo Metal Mining

镍资源和金属材料体系完整，历史上与 MLCC 镍电极粉链条相关；缺点是公司体量较大，资源、冶炼和材料业务会稀释 MLCC 单线弹性。

MLCC Materials · 镍粉 · 5727.T · Toho Titanium

金属粉体相关标的，历史上出现在 MLCC 电极粉供应商名单中；当前重点不是静态持仓，而是持续核查 JX Advanced Metals 整合及交易状态。

Networking · MRVL · Marvell

数据中心网络与互联芯片标的。适合观察 Networking +121% 是否从概念转化为订单与收入增长。

Networking · AVGO · Broadcom

网络 ASIC 与交换芯片核心。Rubin 机柜级互联升级，会强化数据中心网络芯片价值。

Networking · NVDA · NVIDIA

Rubin 平台锚。其规格、量产时间、NVLink 设计和生态推进速度决定整条链的估值节奏。

ABF · 8046.TW · Nan Ya PCB / 南电

ABF 载板核心观察。与欣兴共同验证高阶载板供需缺口和报价弹性。

Power / Cooling · 2308.TW · Delta / 台达电

电源、供配电与散热系统平台型标的。冷板纯度不如健策、奇鋐，但在电源、供配电和 cooling 系统整合上更完整。

Off-rack · VRT · Vertiv

数据中心电力和热管理基础设施龙头。它不是机柜内 BOM 标的，而是 AI 数据中心供电、UPS、配电和热管理扩建的机柜外映射。

Cooling · 3653.TW · Jentech Precision / 健策

精密金属件与高性能热管理标的，cold plate 映射更纯。适合放在冷板/液冷链条优先观察位。

Cooling · 3017.TW · AVC / 奇鋐

散热模组与液冷供应链纯度高，上市可投资性好。适合验证 Rubin 热密度提升是否外溢到整套液冷系统。

Off-rack · 6805.TW · 富世达

水冷机构件和快接头观察标的。更偏液冷细分零组件弹性，适合作为机柜外液冷基础设施扩散指标。

七、结论

Rubin 相对 Blackwell 的投资含义，是 AI 机柜价值从 GPU 单点向 Memory、PCB、MLCC、Networking、ABF、电源与散热系统扩散。最强的价格弹性来自 Memory、PCB、MLCC；PCB 内部最关键的新增项是 Midplane 从无到有，Unimicron 的确定性、ZDT 的高弹性、深南电路的 PCB + 载板 + 装联协同和胜宏科技的份额结构风险需要分开跟踪；NVDA 仍是平台节奏锚。
